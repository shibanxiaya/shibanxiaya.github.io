# 关于本站

本博客使用 **Jekyll** 静态网站生成器搭建，托管于 **GitHub Pages**。

## 搭建过程

本站通过以下技术栈搭建：

| 技术 | 用途 |
|------|------|
| Jekyll | 静态网站生成框架 |
| Docker | 容器化开发环境 |
| Markdown | 内容编写 |
| GitHub Pages | 网站托管与自动部署 |
| Minima | Jekyll 默认主题 |

## 实验说明

本站同时也是**实验三 Jekyll 静态网站生成**课程的作业成果，完整展示了：

1. Jekyll 站点搭建流程（Docker 环境）
2. Markdown 内容编写与发布
3. GitHub Pages 部署与访问
