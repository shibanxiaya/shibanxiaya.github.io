## 技术路线

### 充电需求预测

```
数据收集 → 特征工程 → LSTM/Transformer建模 → 结果分析
```

- 数据源：充电站历史数据
- 方法：时序预测（LSTM、Transformer）
- 目标：准确预测区域充电需求峰值

### NLP 模型对比

```
数据预处理 → 模型构建(MLP/CNN/LSTM/Transformer) → 训练评估 → 对比分析
```

- 任务1：情感分类（二分类）
- 任务2：命名实体识别（序列标注）
- 评价指标：准确率、F1分数
